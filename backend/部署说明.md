# 🌈 彩虹创口贴 - 后端部署快速指南

## 方式一：一键部署（最简单）⭐

```bash
# 1. 上传所有文件到服务器
scp -r * root@你的服务器 IP:/opt/rainbow-helper/

# 2. SSH 登录服务器
ssh root@你的服务器 IP

# 3. 进入目录
cd /opt/rainbow-helper

# 4. 执行一键部署脚本
bash deploy.sh

# 5. 修改 .env 文件中的 API 密钥（如果需要）
nano .env
# 按 Ctrl+X, Y, Enter 保存

# 6. 重启服务
systemctl restart rainbow-helper
```

---

## 方式二：手动部署

```bash
# 1. 上传文件
scp -r * root@你的服务器 IP:/opt/rainbow-helper/

# 2. 登录服务器
ssh root@你的服务器 IP
cd /opt/rainbow-helper

# 3. 安装依赖
apt update && apt install python3 python3-pip -y
pip3 install -r requirements.txt

# 4. 配置环境变量
cat > .env << EOF
QWEN_API_KEY=你的 API 密钥
SERVER_PORT=7860
EOF

# 5. 配置 systemd 服务
cp rainbow-helper.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable rainbow-helper
systemctl start rainbow-helper

# 6. 查看状态
systemctl status rainbow-helper
```

## 开放防火墙

### 阿里云/腾讯云
在控制台安全组添加入站规则：
- 端口范围：7860/7860
- 授权对象：0.0.0.0/0

### Ubuntu UFW
```bash
ufw allow 7860/tcp
ufw enable
```

## 前端配置

后端部署完成后，修改前端 `.env` 文件：

```bash
# Rainbow_Helper/client/.env
VITE_API_BASE_URL=http://你的服务器 IP:7860
```

然后打包前端：

```bash
cd client
npm run build
```
