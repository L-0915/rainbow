# 🌈 彩虹创口贴 - 云服务器部署完整指南

## 📋 快速部署（推荐 systemd 方式）

### 1. 上传文件到服务器

```bash
# 在本地执行，上传所有文件
scp -r * root@你的服务器IP:/opt/rainbow-helper/
```

### 2. 登录服务器并安装依赖

```bash
# SSH 登录服务器
ssh root@你的服务器IP

# 进入目录
cd /opt/rainbow-helper

# 更新系统包
apt update && apt upgrade -y

# 安装 Python3 和 pip（如果未安装）
apt install python3 python3-pip -y

# 安装 Python 依赖
pip3 install -r requirements.txt

# 安装 Node.js（如果需要重新构建前端）
curl -fsSL https://deb.nodesource.com/setup_20.x | bash -
apt install nodejs -y
```

### 3. 配置环境变量

```bash
# 创建 .env 文件
cat > .env << EOF
QWEN_API_KEY=你的千问 API 密钥
QWEN_API_URL=https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
SERVER_PORT=7860
EOF
```

### 4. 部署 systemd 服务（推荐）

```bash
# 复制服务文件
cp rainbow-helper.service /etc/systemd/system/

# 重新加载 systemd
systemctl daemon-reload

# 启用服务（开机自启）
systemctl enable rainbow-helper

# 启动服务
systemctl start rainbow-helper

# 查看状态
systemctl status rainbow-helper

# 查看日志
journalctl -u rainbow-helper -f
```

### 5. 开放防火墙端口

```bash
# 如果使用 UFW
ufw allow 7860/tcp
ufw allow 80/tcp
ufw allow 443/tcp
ufw enable

# 如果使用阿里云/腾讯云，需在控制台安全组开放端口
```

## 🔄 备用方案：使用保活脚本

如果不想用 systemd，可以使用保活脚本：

```bash
# 给脚本添加执行权限
chmod +x start_server.sh

# 运行脚本
./start_server.sh

# 添加到 crontab 每分钟检查（可选）
crontab -e
# 添加下面这行：
* * * * * /opt/rainbow-helper/start_server.sh > /dev/null 2>&1
```

## 🔧 故障排查

### 服务无法启动

```bash
# 检查端口占用
netstat -tlnp | grep 7860

# 检查 Python 版本
python3 --version

# 手动运行测试
python3 app.py
```

### 查看日志

```bash
# systemd 日志
journalctl -u rainbow-helper --since "1 hour ago"

# 应用日志
tail -100 /opt/rainbow-helper/logs/server.log
tail -100 /opt/rainbow-helper/logs/error.log
```

## 🔐 安全建议

### 1. 配置 HTTPS（推荐）

```bash
# 安装 Certbot
apt install certbot python3-certbot-nginx -y

# 获取证书（需要域名）
certbot --nginx -d your-domain.com

# 自动续期
certbot renew --dry-run
```

### 2. 配置 Nginx 反向代理

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 强制 HTTPS
    if ($scheme = http) {
        return 301 https://$server_name$request_uri;
    }

    location / {
        proxy_pass http://127.0.0.1:7860;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

### 3. 定期备份数据库

```bash
# 添加定时任务
crontab -e

# 每天凌晨 3 点备份数据库
0 3 * * * cp /opt/rainbow-helper/emotion_data.db /opt/backups/emotion_data_$(date +\%Y\%m\%d).db
```

## 📱 前端打包配置

后端部署完成后，配置前端 API 地址：

```bash
# 编辑 client/.env
VITE_API_BASE_URL=https://你的域名.com
# 或者直接用 IP
VITE_API_BASE_URL=http://你的服务器 IP:7860
```

然后打包前端：

```bash
cd client
npm install
npm run build
```

将构建产物 `client/dist/` 目录上传到服务器的 `static/` 目录。
