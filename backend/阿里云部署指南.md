## 🚀 阿里云部署步骤

### 1. 登录阿里云容器镜像服务 ACR

1. 打开阿里云控制台
2. 进入 **容器镜像服务 ACR**
3. 创建命名空间（如 `rainbow`）
4. 创建镜像仓库（如 `rainbow-helper`）

### 2. 构建并推送镜像

```bash
# 在本地项目目录执行
cd D:\parent say\彩虹创口贴\Rainbow_Helper

# 构建 Docker 镜像
docker build -t rainbow-helper:latest .

# 标记为阿里云镜像
# 替换为你自己的阿里云 ACR 地址
docker tag rainbow-helper:latest registry.cn-<region>.aliyuncs.com/<namespace>/rainbow-helper:latest

# 登录阿里云（使用 Docker 登录密码）
docker login --username=<你的阿里云账号> registry.cn-<region>.aliyuncs.com

# 推送镜像
docker push registry.cn-<region>.aliyuncs.com/<namespace>/rainbow-helper:latest
```

### 3. 部署到 ECS 云服务器

#### 方式 A：直接使用 ECS（推荐）

```bash
# 1. SSH 登录阿里云 ECS
ssh root@你的 ECS IP

# 2. 安装 Docker（如果未安装）
curl -fsSL https://get.docker.com | bash
systemctl enable docker
systemctl start docker

# 3. 拉取镜像并运行
docker run -d \
  --name rainbow-helper \
  --restart always \
  -p 7860:7860 \
  -v /opt/rainbow-data:/app/data \
  -e SERVER_PORT=7860 \
  harbor.baai.ac.cn/external-cooperation/flagrelease:flagrelease_nv_qwen3_8b \
  python3 app.py
```

#### 方式 B：使用阿里云 ACK（容器服务）

1. 进入 **容器服务 Kubernetes 版**
2. 创建集群或选择现有集群
3. 创建应用，选择镜像 `harbor.baai.ac.cn/external-cooperation/flagrelease:flagrelease_nv_qwen3_8b`
4. 配置端口：容器端口 7860
5. 配置环境变量：
   ```
   SERVER_PORT=7860
   ```
6. 配置存储卷（持久化数据）
7. 配置服务暴露（公网访问）

## 📱 前端配置

### 开发环境配置

创建 `client/.env.development`:

```bash
# 本地开发
VITE_API_BASE_URL=http://localhost:7860
```

### 生产环境配置

创建 `client/.env.production`:

```bash
# 生产环境 - 替换为你的阿里云公网 IP 或域名
VITE_API_BASE_URL=http://你的阿里云公网 IP:7860
```

### 打包前端

```bash
cd client
npm install
npm run build
```

打包产物在 `client/dist/` 目录。

## 📝 注意事项

1. **镜像内置模型**：该镜像可能使用本地模型推理，不需要 API 密钥
2. **GPU 要求**：Qwen 模型可能需要 GPU，确保 ECS 有 GPU 实例
3. **内存要求**：建议至少 16GB 内存
4. **存储持久化**：使用 `-v` 挂载数据卷保存数据库
