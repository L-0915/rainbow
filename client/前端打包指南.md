# 🌈 彩虹创口贴 - 前端打包成 App 指南

## 📋 前提条件

### Node.js
确保已安装 Node.js 16+：
```bash
node --version
```

### Java JDK（Android 打包需要）
- 下载：https://adoptium.net/
- 配置环境变量 `JAVA_HOME`

### Android Studio（可选，用于调试）
- 下载：https://developer.android.com/studio

---

## 🚀 快速打包步骤

### 1. 安装依赖

```bash
cd client
npm install
```

### 2. 配置后端 API 地址

编辑 `client/.env.production`：

```bash
# 替换为你的后端服务器地址
VITE_API_BASE_URL=http://你的阿里云公网 IP:7860
```

### 3. 构建前端

```bash
npm run build
```

构建产物在 `client/dist/` 目录。

### 4. 添加 Android 平台

```bash
# 第一次需要添加
npx cap add android
```

### 5. 同步代码到 Android

```bash
npx cap sync
```

### 6. 打开 Android Studio

```bash
npx cap open android
```

### 7. 在 Android Studio 中打包

1. **打开项目**：`client/android`
2. **等待 Gradle 同步完成**
3. **构建 APK**：
   - 菜单：`Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
4. **APK 位置**：`client/android/app/build/outputs/apk/debug/app-debug.apk`

## 📱 生成 Release APK（签名版）

### 1. 生成签名密钥

```bash
# 在 client/android 目录执行
keytool -genkey -v -keystore rainbow.keystore -alias rainbow -keyalg RSA -keysize 2048 -validity 10000
```

### 2. 配置签名

编辑 `client/android/app/build.gradle`：

```gradle
android {
    ...
    signingConfigs {
        release {
            storeFile file("rainbow.keystore")
            storePassword "你的密钥密码"
            keyAlias "rainbow"
            keyPassword "你的密钥密码"
        }
    }
    buildTypes {
        release {
            signingConfig signingConfigs.release
            minifyEnabled true
            shrinkResources true
        }
    }
}
```

### 3. 构建 Release APK

在 Android Studio 中：
- `Build` → `Generate Signed Bundle / APK`
- 选择 `APK`
- 选择 `release` 构建类型
- 完成

## 🔧 常见问题

### 1. npx cap add android 失败

```bash
# 清理缓存
rm -rf android
npm install
npx cap add android
```

### 2. Gradle 同步失败

- 检查网络连接
- 检查 Java 版本（需要 JDK 11 或 17）
- 在 Android Studio 中：`File` → `Invalidate Caches`

### 3. APK 无法安装

- 确保手机允许"未知来源"
- 检查 Android 版本兼容性（最低 API 22）

### 4. 网络请求失败

编辑 `client/android/app/src/main/AndroidManifest.xml`，确保有网络权限：

```xml
<uses-permission android:name="android.permission.INTERNET" />
<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />
```

## 📲 APK 安装

### 方式 1：USB 传输
1. 将 APK 复制到手机
2. 在文件管理器中点击安装

### 方式 2：扫码下载
1. 将 APK 上传到服务器
2. 手机扫码下载安装

### 方式 3：Android Debug Bridge
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
