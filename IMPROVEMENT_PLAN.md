## 二、用户体验优化

### 2.1 情绪可视化增强
**现状**：有情绪记录功能，但可视化展示不够直观

**改进方案**：
- 情绪日历：日历视图展示每日情绪，不同颜色代表不同心情
- 情绪趋势图：周/月维度展示情绪变化曲线
- 情绪统计：本周最常心情、心情变化规律

**文件**：`client/src/components/emotion/EmotionChart.tsx`（增强）

### 2.2 每日提醒系统
**新增功能**：
- 定时提醒孩子记录心情（如每天晚上8点）
- 温馨的推送文案（"小彩虹想你了～今天心情怎么样呀？🌈"）
- 家长端设置提醒时间

**技术方案**：使用 Capacitor Local Notifications 插件

### 2.3 成就系统完善
**现状**：有成就系统框架

**增强方案**：
- 成就可视化：徽章墙展示已解锁成就
- 解锁动画：获得成就时的庆祝效果
- 成就分类：情绪类、社交类、成长类

**文件**：`client/src/store/appStore.ts`（扩展成就数据）

### 2.4 引导流程优化
**新增**：
- 首次使用引导页（3-4页滑动引导）
- 功能发现提示（气泡提示）
- 年龄选择（3-6岁/7-9岁/10-12岁，调整UI风格）

## 三、功能完善

### 3.1 AI 语音输出
**现状**：AI 只能文字回复

**新增**：
- 小彩虹语音播报功能
- 使用 Web Speech API 或第三方 TTS
- 可开关语音播报

**文件**：新建 `client/src/hooks/useSpeechSynthesis.ts`

### 3.2 漂流瓶功能优化
**现状**：有漂流瓶功能

**优化**：
- 漂流瓶动画效果
- 捡到瓶子时的惊喜效果
- 内容审核与过滤

### 3.3 家长端数据导出
**新增**：
- 导出孩子情绪报告（PDF/图片）
- 分享功能
- 数据统计面板

**文件**：新建 `client/src/components/parent/ReportExport.tsx`

### 3.4 多账号切换
**新增**：
- 支持多个孩子账号
- 家长可切换查看不同孩子数据
- 本地数据隔离
